# Smart Lock System

## Overview

Smart Lock System is a security-focused project developed as part of a college project. It consists of an Android application and a Python prototype that demonstrate application-level security using PIN authentication, intruder detection, photo capture, and email notifications.

The Android application provides the complete user interface and security features, while the Python prototype demonstrates the core detection and alert mechanism.

---

## Features

### Android Application

- PIN-based authentication
- Application locking
- Intruder photo capture
- Intruder gallery
- Email notification support
- Security event logging
- Alarm on multiple failed attempts
- Settings management

### Python Prototype

- Face detection
- Photo capture
- Email alerts
- OpenCV integration
- MediaPipe integration

## Installation

### Android Application

1. Clone the repository.
2. Open the project in Android Studio.
3. Sync Gradle.
4. Build and run the application.

### Python Prototype

1. Navigate to the `Python-Prototype` directory.
2. Install the required dependencies.
3. Run the main Python file.
